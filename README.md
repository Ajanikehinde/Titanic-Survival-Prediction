# Titanic-Survival-Prediction
This project applies machine learning techniques to predict passenger survival on the Titanic. It follows a structured pipeline covering data exploration, preprocessing, modeling, evaluation, and final prediction, with an emphasis on reproducibility and professional documentation.
# Titanic Survival Prediction Project

## Problem Statement
Predict whether a passenger survived the Titanic disaster using machine learning.

## Dataset Overview
Source: Kaggle Titanic Dataset  
Features include passenger demographics, ticket class, fare, and family structure.

## Approach
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Modeling
5. Evaluation
6. Final Prediction

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Results
Random Forest achieved the best performance based on accuracy and F1-score.

## Tools Used
Python, Pandas, NumPy, Seaborn, Scikit-learn, Jupyter Notebook
